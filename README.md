<h2>Lições Aprendidas</h2>

Primeiro projeto realizado na Trybe, com o intuito de montar um portfólio simples utilizando os conceitos aprendidos de HTML e CSS.
Os principais desafios foram: implementar uma barra superior fixa com um título, adicionar: uma foto, um artigo sobre mim e duas lista de lições já aprendidas e lições a aprender.

<h3>Tecnologias utilizadas</h3>

Front-end: HTML e CSS

<h3>Instalando Dependências</h3>

Em seu terminal:</br> 
cd licoes-aprendidas/</br>
npm install ou npm i

<h3>Visualizando em seu navegador</h3>

Em seu terminal:</br>
cd licoes-aprendidas/</br>
code .</br>
Clique na opção "Go Live" no canto inferior direito

<h3>Rodando os testes da aplicação</h3>

Em seu terminal:</br>
cd licoes-aprendidas/</br>
npm run cypress:open
